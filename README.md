# aspnet.foundation

A Docker (Windows) container that supports running ASP.net Framework 
apps using Web Forms (aspx) or WCF (asmx) 

Due to some issues with the accessibility of the repos for the Roslyn compilers
they are currently being pushed in via copy commands. 

Source is available at: https://github.com/Druware/aspnet.foundation
